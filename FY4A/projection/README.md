### 风云4A卫星全圆盘标称网格行列号与地理经纬度的相互转换
#### fy4a_projection_calculator.py  计算法实现
#### fy4a_projection_lookup.py  查表法实现
#### main.ipynb 一些测试，结论有：
    1. 行列号到经纬度，计算法得到的跟查表法得到的误差极小，可认为一样，误差-13左右数量级
    2. 经纬度到行列号，计算法得到的跟查表法（球面距离最小）得到的，差别不大，在中国区最多差一个像素；但是速度快太多
    3. 查表法尝试用（球面距离最小）以及（先查最近纬度再在那行查最近经度），发现后者误差很大
    4. 综上所述，经纬度到行列号建议用计算法，行列号到经纬度两者都可
    5. 注意：这个支持多种分辨率，但都是针对全圆盘的
